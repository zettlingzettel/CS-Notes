How to install tmux on ubuntu

Be under the root directory
’apt install tmux’

If waiting for cache lock 3321
sudo kill 3321
sudo kill -9 3321

To run tmux
’tmux’

To detach tmux
’ctrl b + d’

Create a new window
’tmux new -s Session1’

Split the window
’ctrl b + %’
